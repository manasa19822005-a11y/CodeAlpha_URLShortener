# URL Shortener

## 📌 Project Description
A URL Shortener web application built using Python and Flask that converts long URLs into short, easy-to-share links.

## 🚀 Features
- Convert long URLs into short URLs
- Redirect users to the original website
- Store URL data in a database
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_URLShortener.git
```

2. Open the project folder

```bash
cd CodeAlpha_URLShortener
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open the browser and visit:

```
http://127.0.0.1:5000
```
